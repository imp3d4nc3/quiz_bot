# Discord クイズBot (Quiz Bot)

Discordサーバーでクイズ大会を開催するためのBotです。
出題者と問題数を指定して開始すると、出題者のDMを通じて問題の登録（対話型）を行い、サーバーのチャンネルでボタン選択式のクイズを出題します。全員の回答が出揃うと自動的に正解と現在のスコアを発表し、全問題が終了すると最終ランキングを出力します。

---

## 主な機能

1. **簡単セットアップ**: `/quiz_start` コマンドで出題者、問題数、出題チャンネルを指定してクイズを開始。
2. **ボタン式回答**: 選択肢（最大25個）に応じたボタンが自動生成され、クリックするだけで回答可能。
3. **DMでの問題・配点登録（対話型）**:
   - 出題者にBotからDMが届きます。
   - `問題文を入力` ➡️ `選択肢をカンマ区切りで入力` ➡️ `正解番号を入力` ➡️ `この問題の配点を入力` の順に送るだけで、簡単に問題が登録されます。
4. **ウルトシステム（得点2倍アイテム）**:
   - プレイヤーはセッション中に **2回まで** 「⚡ウルトを発動」ボタンを押すことができます。
   - ウルトを使用し、正解した場合、その問題の得点（配点）が **2倍** になります。
5. **後半戦の救済措置**:
   - クイズの残り問題数が **25%以下** になった時点で、現時点で **ワースト2位**（下から2番目のスコア）のプレイヤー全員に、ウルトの使用権が **1回分追加付与** されます。
6. **自動＆手動締め切り**:
   - 参加登録した全員が回答すると、自動で回答を締め切って正解を発表。
   - 未回答者がいる場合などは、出題者が「回答を締め切る」ボタンを押して手動で締め切ることも可能です。
7. **2つの部門別ランキング出力**:
   - すべての問題が終了すると、配点やウルト（得点2倍）を考慮した **「ポイント部門」** と、純粋な正解数を競う **「正答率部門」** の2つのランキングに分けてリザルトが発表されます。

## セットアップ手順

### 1. Discord Developer Portal での準備
1. [Discord Developer Portal](https://discord.com/developers/applications) にアクセスし、新しいアプリケーションを作成します。
2. 左メニューの **Bot** セクションに移動し、Botを作成します。
3. **Privileged Gateway Intents**（特権インテント）のエリアで、以下の項目を **ON** にします：
   - **PRESENCE INTENT** (任意)
   - **SERVER MEMBERS INTENT** (参加者の取得に必要)
   - **MESSAGE CONTENT INTENT** (DM等でのメッセージ内容読み取りに必要)
4. 左メニューの **OAuth2** -> **URL Generator** に移動し、以下を選択します：
   - **Scopes**: `bot`, `applications.commands`
   - **Bot Permissions**: `Send Messages`, `Embed Links`, `Read Message History`, `Use External Emojis` (通常はデフォルトのテキスト権限があれば十分です)
5. 生成されたURLを使用して、BotをあなたのDiscordサーバーに招待します。

### 2. 環境変数の設定
プロジェクトのルートディレクトリにある `.env` ファイルを開き、設定を行います。

```env
DISCORD_TOKEN=ここに取得したBotのトークンを貼り付け
GUILD_ID=ここにテストするDiscordサーバーのID（ギルドID）を貼り付け
```

> [!TIP]
> Discordのスラッシュコマンドはグローバルに反映するまでに最大1時間ほどかかります。`.env` ファイルに `GUILD_ID` を指定しておくと、Botの起動時に対象サーバー（ローカル）に即座にスラッシュコマンドが反映されます（開発・テスト時に推奨）。


### 3. ライブラリのインストール
以下のコマンドを実行して、必要なパッケージをインストールします。

```bash
pip install -r requirements.txt
```

## 起動方法

Botを起動するには、以下の **2つの方法** があります。

### 方法1：起動用アプリケーションを使う（推奨）
プロジェクトフォルダ内にある **`run_launcher.bat`** をダブルクリックして起動します。

* **設定の管理**: アプリケーション画面上で `DISCORD_TOKEN`（トークン）や `GUILD_ID`（サーバーID）を入力して「設定を保存」を押すことで、`.env` ファイルが自動で更新・保存されます。
* **Botの起動・停止**: 「▶ Botを起動する」ボタンでBotが開始し、画面上の「コンソールログ」にリアルタイムで動作ログが出力されます。「■ Botを停止する」ボタンで安全にBotを停止できます。

### 方法2：コマンドラインを使う
コマンドプロンプトやPowerShellで以下のコマンドを実行してBotを起動します。

```bash
python bot.py
```

Botがログインに成功すると、コンソールに `Logged in as Bot名` と表示され、スラッシュコマンドが同期されます。

## クイズの遊び方

1. **クイズの開始**:
   チャンネルで `/quiz_start` コマンドを実行します。
   - 引数 `host`: 出題者にするメンバーを選択
   - 引数 `questions`: クイズの問題数（例: `3`）
   - 引数 `channel`: (任意) 出題を行うチャンネル。指定しない場合はコマンドを実行したチャンネルになります。

2. **参加表明**:
   チャンネルに「参加受付中」のメッセージが表示されます。回答したいプレイヤーは **「参加する」** ボタンを押します。
   全員の参加が完了したら、出題者が **「参加受付を締め切る (出題者用)」** ボタンを押します。

3. **問題の登録 (出題者のDM)**:
   受付が締め切られると、出題者のDMにBotからメッセージが届きます。
   1. **問題文**を送信します。
   2. 選択肢をカンマ区切り（例: `織田信長, 豊臣秀吉, 徳川家康`）で送信します。
   3. 正解の番号（例: `1`）を半角数字で送信します。
   4. この問題の**配点**（例: `3`）を半角数字で送信します。

4. **出題と回答**:
   登録が完了すると、指定したチャンネルにボタン付きで問題が出力されます。
   - 参加者は正しいと思う選択肢番号のボタンを押して回答します。
   - 回答する前に **「⚡ウルトを発動」** ボタンを押すと、その問題で正解した際の得点が2倍になります（1セッションに初期で2回まで）。
   - クイズ進行が後半（残り問題数が25%以下）に突入した際、現時点でワースト2位のプレイヤー全員にウルトが1個追加付与されます。

5. **正解発表**:
   参加者全員が回答するか、出題者が「回答を締め切る」ボタンを押すと、自動的に正解が発表され、ウルト使用者の結果（2倍獲得など）や現在のスコア一覧が表示されます。
   5秒後、次の問題の登録（DM）へ進みます。

6. **ランキング発表**:
   設定した問題数に達すると、最終結果が「ポイント部門」と「正答率部門」の2つに分かれて発表され、クイズセッションが終了します。



7. **クイズの強制終了**:
   もしクイズを途中で中止したい場合は、チャンネルで `/quiz_abort` コマンドを実行します（出題者または管理者のみ実行可能）。
